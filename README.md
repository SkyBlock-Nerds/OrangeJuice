## How to run project
1. For the time being we are using the local maven repo version of nerdbot-generator so make sure you have that in your maven repo <br />
    In case you don't:
   1. Clone the [NerdBot repo](https://github.com/skyBlock-Nerds/NerdBot)
   2. run `mvn clean install` in NerdBot the repo dir
    > [!WARNING]
    > Make sure you pull and run `mvn clean install` when there are changes in the NerdBot generator that you want to have.
